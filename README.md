# AgriVision — How Drones Are Redefining Modern Farming

A single-page scrolling website about agricultural drones and computer vision
technology, built for a general public audience. Inspired by the clean, modern
look of the DJI Agriculture website.

## Stack

- React 18 + TypeScript
- Vite

## Run it

```bash
npm install
npm run dev
```

Then open the URL Vite prints (usually `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```

## Sections

1. **Hero** — Bold headline introduction with a drone-over-field background.
2. **The Problem** — Labor shortages, crop disease, resource waste.
3. **How It Works** — Computer vision, deep learning, and CNNs explained
   simply, with a flow diagram.
4. **Industry Impact** — Bold stat callouts (95% accuracy, targeted spraying,
   automated scouting, livestock health).
5. **Real World Use Cases** — Disease detection, weed detection, livestock
   monitoring.
6. **Navigation & Autonomous Drones** — Hyperspectral imaging and
   stereo-vision mapping.
7. **The Future** — Sustainability, affordability, food security.
8. **Footer** — Sources / citations.

## Notes

Imagery is loaded from Unsplash CDN (no local assets to manage). Replace the
URLs in the section components if you need offline assets.
